# Delphi_REST_Integration
Integração de sistema legado Delphi com serviços modernos REST
Objetivo: mostrar capacidade de modernizar sistemas legados
Funcionalidades: CRUD via REST, parsing JSON, logs de erro
“Implementação completa da integração e testes de API”

Este projeto é uma demonstração baseada em experiências reais no desenvolvimento de sistemas corporativos, com código e dados totalmente recriados para fins de portfólio.
